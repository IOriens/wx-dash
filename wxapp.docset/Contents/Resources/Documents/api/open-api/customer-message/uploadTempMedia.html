uploadTempMedia

把媒体文件上传到微信服务器。目前仅支持图片。用于发送客服消息或被动回复用户消息。

请求地址

POST https://api.weixin.qq.com/cgi-bin/media/upload?access_token=ACCESS_TOKEN&type=TYPE

参数

string access_token

接口调用凭证

string type

文件类型

type 的合法值

值 说明
image 图片
file media

form-data 中媒体文件标识，有filename、filelength、content-type等信息

返回值

Object

返回的 JSON 数据包

属性 类型 说明 支持版本
errcode number 错误码
errmsg string 错误信息
type string 文件类型
media_id string 媒体文件上传后，获取标识
created_at number 媒体文件上传时间戳

errcode 的合法值

值 说明
40004 无效媒体文件类型

type 的合法值

值 说明
image 图片

调用示例

使用 CURL 命令，用 FORM 表单方式上传一个多媒体文件

curl -F media=@test.jpg "https://api.weixin.qq.com/cgi-bin/media/upload?access_token=ACCESS_TOKEN&type=TYPE"

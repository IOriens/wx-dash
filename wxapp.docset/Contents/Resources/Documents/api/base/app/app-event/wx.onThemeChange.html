# wx.onThemeChange(function callback)

基础库 2.11.0 开始支持，低版本需做兼容处理。

监听系统主题改变事件。该事件与 App.onThemeChange 的回调时机一致。

# 参数

# function callback

系统主题改变事件的回调函数

# 参数

# Object res
属性 类型 说明
theme string 系统当前的主题，取值为light或dark

theme 的合法值

值 说明 最低版本
'dark' 深色主题
'light' 浅色主题

# 注意

  • 只有在全局配置"darkmode": true时才会触发此事件。

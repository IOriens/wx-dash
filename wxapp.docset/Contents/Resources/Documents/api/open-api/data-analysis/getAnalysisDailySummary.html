getAnalysisDailySummary

获取用户访问小程序数据概况

请求地址

POST https://api.weixin.qq.com/datacube/getweanalysisappiddailysummarytrend?access_token=ACCESS_TOKEN

参数

string access_token

接口调用凭证

string begin_date

开始日期。格式为 yyyymmdd

string end_date

结束日期，限定查询1天数据，允许设置的最大值为昨日。格式为 yyyymmdd

返回值

Object

返回的 JSON 数据包

属性 类型 说明 支持版本
list Array.<Object> 数据列表

list 的结构

属性 类型 说明 支持版本
ref_date string 日期，格式为 yyyymmdd
visit_total number 累计用户数
share_pv number 转发次数
share_uv number 转发人数

请求数据示例

{
  "begin_date" : "20170313",
  "end_date" : "20170313"
}

返回数据示例

{
  "list": [
    {
      "ref_date": "20170313",
      "visit_total": 391,
      "share_pv": 572,
      "share_uv": 383
    }
  ]
}

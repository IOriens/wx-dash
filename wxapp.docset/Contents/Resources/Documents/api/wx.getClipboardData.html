wx.getClipboardData(Object object)

基础库 1.1.0 开始支持，低版本需做兼容处理。

获取系统剪贴板的内容

参数

Object object
属性 类型 默认值 必填 说明
success function 否 接口调用成功的回调函数
fail function 否 接口调用失败的回调函数
complete function 否 接口调用结束的回调函数（调用成功、失败都会执行）
object.success 回调函数

参数

Object object

属性 类型 说明
data string 剪贴板的内容

示例代码

wx.getClipboardData({
  success(res) {
    console.log(res.data)
  }
})

# BLEPeripheralServer.writeCharacteristicValue(Object Object)

基础库 2.10.3 开始支持，低版本需做兼容处理。

往指定特征值写入数据，并通知已连接的主机，从机的特征值已发生变化，该接口会处理是走回包还是走订阅。

# 参数

# Object Object

属性 类型 默认值 必填 说明
serviceId String 是 service 的 uuid
characteristicId String 是 characteristic对应的uuid
value ArrayBuffer 是 特征值对应的二进制值
needNotify Boolean 是 是否需要通知主机value已更新
callbackId Number 否 可选，处理回包时使用

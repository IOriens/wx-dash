# wx.setStorageSync(string key, any data)

本接口从基础库版本 1.9.6 起支持在小程序插件中使用

wx.setStorage 的同步版本

# 参数

# string key

本地缓存中指定的 key

# any data

需要存储的内容。只支持原生类型、Date、及能够通过JSON.stringify序列化的对象。

# 示例代码

wx.setStorage({
  key:"key",
  data:"value"
})

try {
  wx.setStorageSync('key', 'value')
} catch (e) { }

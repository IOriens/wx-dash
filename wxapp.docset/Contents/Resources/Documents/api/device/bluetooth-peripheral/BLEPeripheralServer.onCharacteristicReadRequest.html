# BLEPeripheralServer.onCharacteristicReadRequest(function callback)

基础库 2.10.3 开始支持，低版本需做兼容处理。

监听已连接的设备请求读当前外围设备的特征值事件。收到该消息后需要立刻调用 writeCharacteristicValue 写回数据，否则主机不会收到响应。

# 参数

# function callback

已连接的设备请求读当前外围设备的特征值事件的回调函数

# 参数

# Object res
属性 类型 说明
serviceId String service对应的uuid
characteristicId String characteristic对应的uuid
callbackId Number 唯一标识码，调用 writeCharacteristicValue 时使用

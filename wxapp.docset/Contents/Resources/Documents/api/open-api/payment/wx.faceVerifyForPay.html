wx.faceVerifyForPay(Object object)

支持版本 >= 2.3.0

支付各个安全场景验证人脸

参数

Object object
属性 类型 默认值 是否必填 说明 支持版本
scene number 是 场景值
package string 是 支付传入的参数
packageSign string 是 支付传入的参数，签名
otherVerifyTitle string 是 别的验证方式woridng，不传就不显示入口
success function 否 接口调用成功的回调函数
fail function 否 接口调用失败的回调函数
complete function 否 接口调用结束的回调函数（调用成功、失败都会执行）
object.success 回调函数

参数

Object res

属性 类型 说明 支持版本
token string 本次认证结果凭据
clickOtherVerifyBtn boolean 用户是否点了别的验证方式
object.fail 回调函数

参数

Object res

属性 类型 说明 支持版本
errMsg string 错误信息
number number 错误码

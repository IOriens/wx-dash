getTemplateLibraryById

获取模板库某个模板标题下关键词库

请求地址

POST https://api.weixin.qq.com/cgi-bin/wxopen/template/library/get?access_token=ACCESS_TOKEN

参数

string access_token

接口调用凭证

string id

模板标题id，可通过接口获取，也可登录小程序后台查看获取

返回值

Object

返回的 JSON 数据包

属性 类型 说明 支持版本
errcode number 错误码
errmsg string 错误信息
id string 模板标题 id
title string 模板标题
keyword_list Array.<Object> 关键词列表

keyword_list 的结构

属性 类型 说明 支持版本
keyword_id string 关键词 id，添加模板时需要
name string 关键词内容
example string 关键词内容对应的示例

POST 数据格式：JSON

请求数据示例

{
 "id": "AT0002"
}

{
 "errcode": 0,

 "title": "购买成功通知",

     {
         "keyword_id": 3,
         "name": "购买地点",
         "example": "TIT造舰厂"
     },
     {
         "keyword_id": 4,
         "name": "购买时间",
         "example": "2016年6月6日"
     },
     {

     }
 ]
}

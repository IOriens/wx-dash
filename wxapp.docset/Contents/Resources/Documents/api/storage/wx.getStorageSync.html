# any wx.getStorageSync(string key)

本接口从基础库版本 1.9.6 起支持在小程序插件中使用

wx.getStorage 的同步版本

# 参数

# string key

本地缓存中指定的 key

# 返回值

# any data

key对应的内容

# 示例代码

wx.getStorage({
  key: 'key',
  success (res) {
    console.log(res.data)
  }
})

try {
  var value = wx.getStorageSync('key')
  if (value) {
    // Do something with return value
  }
} catch (e) {
  // Do something when catch error
}

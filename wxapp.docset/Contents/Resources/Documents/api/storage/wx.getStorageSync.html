Object|string wx.getStorageSync(string key)

wx.getStorage 的同步版本

参数

string key

本地缓存中指定的 key

返回值

Object|string data

key对应的内容

示例代码

wx.getStorage({
  key: 'key',
  success (res) {
    console.log(res.data)
  }
})

try {
  var value = wx.getStorageSync('key')
  if (value) {
    // Do something with return value
  }
} catch (e) {
  // Do something when catch error
}

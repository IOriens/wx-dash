# UDPSocket.onMessage(function callback)

监听收到消息的事件

# 参数

# function callback

收到消息的事件的回调函数

# 参数

# Object res
属性 类型 说明
message ArrayBuffer 收到的消息
remoteInfo Object 消息来源的结构化信息

remoteInfo 的结构

属性 类型 说明
address string 发送消息的 socket 的地址
family string 使用的协议族，为 IPv4 或者 IPv6
port number 端口号
size number message 的大小，单位：字节

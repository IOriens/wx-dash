# wx.previewMedia(Object object, boolean showmenu)

基础库 2.12.0 开始支持，低版本需做兼容处理。

本接口从基础库版本 2.15.0 起支持在小程序插件中使用

预览图片和视频。

# 参数

# Object object

属性 类型 默认值 必填 说明
sources Array.<Object> 是 需要预览的资源列表
current number 0 否 当前显示的资源序号
success function 否 接口调用成功的回调函数
fail function 否 接口调用失败的回调函数
complete function 否 接口调用结束的回调函数（调用成功、失败都会执行）

object.sources 的结构

属性 类型 默认值 必填 说明
url String 是 图片或视频的地址
type String image 否 资源的类型，默认为图片
poster string 否 视频的封面图片

type 的合法值

值 说明 最低版本
image 图片
video 视频

# boolean showmenu

基础库 2.13.0 开始支持，低版本需做兼容处理。

是否显示长按菜单

# wx.playVoice(Object object)

本接口从基础库版本 1.9.6 起支持在小程序插件中使用

从基础库 1.6.0 开始，本接口停止维护，请使用 wx.createInnerAudioContext 代替

开始播放语音。同时只允许一个语音文件正在播放，如果前一个语音文件还没播放完，将中断前一个语音播放。

# 参数

# Object object

属性 类型 默认值 必填 说明 最低版本
filePath string 是 需要播放的语音文件的文件路径 (本地路径)
duration number 60 否 指定播放时长，到达指定的播放时长后会自动停止播放，单位：秒 1.6.0
success function 否 接口调用成功的回调函数
fail function 否 接口调用失败的回调函数
complete function 否 接口调用结束的回调函数（调用成功、失败都会执行）

# 示例代码

wx.startRecord({
  success (res) {
    const tempFilePath = res.tempFilePath
    wx.playVoice({
      filePath: tempFilePath,
      complete () { }
    })
  }
})

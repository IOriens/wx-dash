wx.chooseMessageFile(Object object)

基础库 2.5.0 开始支持，低版本需做兼容处理。

从客户端会话选择文件。

参数

Object object
属性 类型 默认值 必填 说明
count number 100 否 最多可以选择的图片张数，可以 0～100
type string 'all' 否 所选的文件的类型
success function 否 接口调用成功的回调函数
fail function 否 接口调用失败的回调函数
complete function 否 接口调用结束的回调函数（调用成功、失败都会执行）

object.type 的合法值

值 说明
all 从所有文件选择
video 只能选择视频文件
image 只能选择图片文件
file 只能选择除图片和视频的文件
object.success 回调函数

参数

Object res

属性 类型 说明
tempFiles Array.<Object> 返回选择的文件的本地临时文件对象数组

res.tempFiles 的结构

属性 类型 说明
path string 本地临时文件路径
size number 本地临时文件大小，单位 B
name string 选择的文件名称
type string 选择的文件类型
time number 选择的文件的会话发送时间，Unix时间戳

type 的合法值

值 说明
video 选择了视频文件
image 选择了图片文件
file 选择了除图片和视频的文件

示例代码

wx.chooseMessageFile({
  count: 10,
  type: 'image',
  success(res) {
    // tempFilePath可以作为img标签的src属性显示图片
    const tempFilePaths = res.tempFilePaths
  }
})

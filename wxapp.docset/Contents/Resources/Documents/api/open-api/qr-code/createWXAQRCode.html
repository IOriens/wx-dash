createWXAQRCode

获取小程序二维码，适用于需要的码数量较少的业务场景。通过该接口生成的小程序码，永久有效，有数量限制，详见获取二维码。

请求地址

POST https://api.weixin.qq.com/cgi-bin/wxaapp/createwxaqrcode?access_token=ACCESS_TOKEN

参数

string access_token

接口调用凭证

string path

扫码进入的小程序页面路径，最大长度 128 字节，不能为空

number width

二维码的宽度，默认 430px，最小 280px，最大 1280px

返回值

Object

返回的 JSON 数据包

属性 类型 说明 支持版本
errcode number 错误码
errmsg string 错误信息

errcode 的合法值

值 说明
45029 生成码个数总和到达最大个数限制

返回值说明

如果调用成功，会直接返回图片二进制内容，如果请求失败，会返回 JSON 格式的数据。

注意

  • POST 参数需要转成 JSON 字符串，不支持 form 表单提交。
  • 接口只能生成已发布的小程序的二维码。开发版的带参二维码可以在开发者工具预览时生成。
  • 与 getWXACode 总共生成的码数量限制为 100,000，请谨慎调用。

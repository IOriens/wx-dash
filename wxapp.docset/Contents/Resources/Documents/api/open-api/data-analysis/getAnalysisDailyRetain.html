getAnalysisDailyRetain

获取用户访问小程序日留存

请求地址

POST https://api.weixin.qq.com/datacube/getweanalysisappiddailyretaininfo?access_token=ACCESS_TOKEN

参数

string access_token

接口调用凭证

string begin_date

开始日期。格式为 yyyymmdd

string end_date

结束日期，限定查询1天数据，允许设置的最大值为昨日。格式为 yyyymmdd

返回值

Object

返回的 JSON 数据包

属性 类型 说明 支持版本
ref_date string 日期
visit_uv_new Object 新增用户留存
visit_uv Object 活跃用户留存

visit_uv_new 的结构

属性 类型 说明 支持版本
key number 标识，0开始，0表示当天，1表示1天后，依此类推，key取值分别是：0,1,2,3,4,5,6,7,14,30
value number key对应日期的新增用户数/活跃用户数（key=0时）或留存用户数（k>0时）

visit_uv 的结构

属性 类型 说明 支持版本
key number 标识，0开始，0表示当天，1表示1天后，依此类推，key取值分别是：0,1,2,3,4,5,6,7,14,30
value number key对应日期的新增用户数/活跃用户数（key=0时）或留存用户数（k>0时）

请求数据示例

{
  "begin_date" : "20170313",
  "end_date" : "20170313"
}

返回数据示例

{
  "ref_date": "20170313",
  "visit_uv_new": [
    {
      "key": 0,
      "value": 5464
    }
  ],
  "visit_uv": [
    {
      "key": 0,
      "value": 55500
    }
  ]
}

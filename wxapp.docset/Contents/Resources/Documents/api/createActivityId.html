createActivityId

本接口应在后端服务器调用，详细说明参见后端API。

创建被分享动态消息的 activity_id。详见动态消息。

请求地址

GET https://api.weixin.qq.com/cgi-bin/message/wxopen/activityid/create?access_token=ACCESS_TOKEN

请求参数

属性 类型 默认值 必填 说明
access_token string 是 接口调用凭证

返回值

Object

返回的 JSON 数据包

属性 类型 说明
activity_id string 动态消息的 ID
expiration_time number activity_id 的过期时间戳。默认24小时后过期。
errcode number 错误码

errcode 的合法值

值 说明
0 请求成功
-1 系统繁忙。此时请开发者稍候再试
42001 access_token 过期

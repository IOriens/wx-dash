# wx.startLocationUpdateBackground(Object object)

基础库 2.8.0 开始支持，低版本需做兼容处理。

调用前需要 用户授权 scope.userLocationBackground

开启小程序进入前后台时均接收位置消息，需引导用户开启授权。授权以后，小程序在运行中或进入后台均可接受位置消息变化。

# 参数

# Object object

属性 类型 默认值 必填 说明
success function 否 接口调用成功的回调函数
fail function 否 接口调用失败的回调函数
complete function 否 接口调用结束的回调函数（调用成功、失败都会执行）

# 注意

  • 安卓微信7.0.6版本，iOS 7.0.5版本起支持该接口
  • 需在app.json中配置requiredBackgroundModes: ['location']后使用
  • 获取位置信息需配置地理位置用途说明。
